# NrSecreto---JS---Front-End
Jogo em JS com API reconhecimento de voz (chrome).

- Função Math.randon() para sortear número.
- InnerHTML para alterar número max-min conforme const.
- Web Speech API para reconhecimento de voz (doc https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API)
- Função para ativar/manter reconhecimento e manipular DOM.
- Condicionates para validar regras/resultado.
